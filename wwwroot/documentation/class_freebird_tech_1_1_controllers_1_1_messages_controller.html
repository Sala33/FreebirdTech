Detailed Description

This class is responsible for controlling the Messages. Some of it's responsabilities are:

  • define context and user ArtistsController
  • show Index Index
  • exhibit message Details Details(int?)
  • Create a ChatViewModel Create(ChatViewModel)
  • Retrieves the message to be edited Edit(int?)
  • Edits a message in the database Edit(int, Message)
  • Deletes a message Delete(int?)
  • Checks the deletion of a message DeleteConfirmed(int)
  • Check if a message exists MessageExists(int)
  • Get the artist GetArtist(string)
  • Get the artist asynchronously GetArtistAsync().
