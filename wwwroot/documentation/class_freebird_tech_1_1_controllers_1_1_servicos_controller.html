Detailed Description

This class is responsible for controlling the servico. Some of it's responsabilities are:

  • define context and user ServicosController
  • show Index Index
  • exhibit servico Details Details(int?)
  • Create a View with a servico CreateAsync()
  • Create a servico Create(Servico)
  • Retrieves the servico to be edited Edit(int?)
  • Edits a servico in the database Edit(int, Servico)
  • Deletes a servico Delete(int?)
  • Checks the deletion of a servico DeleteConfirmed(int)
  • Check if a servico exists ServicoExists(int)
  • Get the artist based on an id ArtistExists(int).
  • Get the artist based on an ownerId ArtistExists(string).
  • Get the artist GetArtist(string)
