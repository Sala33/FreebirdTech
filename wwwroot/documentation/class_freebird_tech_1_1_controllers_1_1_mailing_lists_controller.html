Detailed Description

This class is responsible for controlling the mailing list. Some of it's responsabilities are:

  • define a contextMailingListsController
  • show Index Index()
  • exhibit mailing list Details Details(int?)
  • creating a mailing list Create
  • saving created mailing list Create(MailingList)
  • edit mailing listEdit(int?)
  • delete mailing listDelete(int?)
  • confirming deletion DeleteConfirmed(int).
  • verify mailing list MailingListExists(int).
