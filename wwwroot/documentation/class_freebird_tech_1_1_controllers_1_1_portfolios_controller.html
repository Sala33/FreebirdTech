Detailed Description

This class is responsible for controlling the porfolios. Some of it's responsabilities are:

  • define context and user ArtistsController
  • show Index Index
  • exhibit portfolio Details Details(int?)
  • Create a View Create()
  • Create a Portfolio Create(Portfolio)
  • Retrieves the portfolio to be edited Edit(int?)
  • Edits a portfolio in the database Edit(int, Portfolio)
  • Deletes a portfolio Delete(int?)
  • Checks the deletion of a portfolio DeleteConfirmed(int)
  • Check if a portfolio exists PortfolioExists(int)
