Detailed Description

This class is responsible for controlling the artist. Some of it's responsabilities are:

  • define context and user ArtistsController
  • show Index Index
  • exhibit artist Details (returns Artist View) Details(int?)
  • get artist and redirect to creation CreateAsync Create(Artist)
  • edit Artist attributes Edit(int, Artist)
  • get current Avatar Image GetCurrentAvatarImage(Artist)
  • delete Artist Delete(int?)
  • confirming deletion DeleteConfirmed(int).
